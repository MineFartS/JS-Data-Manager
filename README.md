# JS Data Manager
A script for easily managing cookies and url parameters in javascript/html
-

Functions:
-
- Parameters
	- data.Param.get()
	- data.Param.set('name', 'value')
	- data.Param.remove('name')
	- data.Param.clear()
- Cookies
	- data.Cookie.get()
	- data.Cookie.set('name', 'value', 'path', 'expire')
	- data.Cookie.remove('name')
	- data.Cookie.clear()
